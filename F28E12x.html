实时处理

  • 160MHz C28x 32位 DSP CPU
  • 等效 320MHz Arm® Cortex®-M7 性能
  • 集成浮点单元 (FPU)

片上内存与安全

  • 高达 128KB Flash (ECC保护)
  • 16KB RAM (奇偶校验保护)
  • JTAGLOCK / Zero-pin boot / Dual-zone security

模拟系统

  • 一个 9.4MSPS, 12位 ADC (17通道)
  • 三个窗口比较器 (CMPSS_LITE)
  • 一个可编程增益放大器 (PGA)

增强型控制外设

  • 8 个总 PWM 通道 (2个 MCPWM)
  • 一个增强型捕获 (eCAP) 模块
  • 一个增强型正交编码器 (eQEP) 模块

通信外设

  • 1x I2C, 1x SPI, 1x UART, 2x SCI
  • 高速 UART 支持 20Mbps

封装与温度

  • 48-pin LQFP, 32-pin LQFP/VQFN
  • 工作结温 (Tj): –40°C 至 125°C
